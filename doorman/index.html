To chat with the doorman

The doorman is a vbrainstem — runs in your browser, calls GitHub Models with your own token. Same auth pattern as the egg hub's summon page; if you've signed in there, you can paste the same token.

  1. Visit github.com/settings/tokens → Generate new token (fine-grained).
  2. Default permissions are fine — GitHub Models is included for any authenticated token.
  3. Paste it below. Stored locally in your browser only (never sent anywhere except GitHub Models + the private companion repo, if any).
